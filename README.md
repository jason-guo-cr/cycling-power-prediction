## 目录说明

- `source-data` 原始数据目录
  - `train` 训练数据目录
  - `test` 测试数据目录
  - `verify` 验证数据目录
  - `rider` 骑行者数据目录
- `parse` 解析fit文件的代码
- `view` 可视化的图片
- `model` 模型文件
- `prediction` 功率预测代码目录


## 功能说明

利用骑行数据来进行功率预测模型的训练

使用fit文件解析出来的csv数据来训练模型

参考上述`目录说明`，根据对应的数据使用xgboost模型来训练功率预测模型

例如可以使用，速度、心率、踏频、距离、时间、坡度、温度等特征来训练模型，当然也可以使用其他特征来训练模型
